Enables your application to send an MMS on behalf of the user.

It returns a String containing the mmsId of the MMS sent The SMSID of the sent SMS is returned in order to ask later for the status of the message as well. The max length of the message must be less than 160 characters.

Parameters:
String$destinationthe address of the recipient of the message
String$textthe text of the message
Strng | null$endpoint(Optional) the endpoint to receive notifications of sent SMSs.
String | null$correlator(Optional) An application generated identifier for the SMS. Mandatory if $endpoint is included. This identifier will be included in the delivery status notifications for your application to correlate notifications with SMS.
Returns:
String. the sent SMS ID
